🚗 Car Price Prediction

This project is a Machine Learning model that predicts the price of used cars based on various features.

📌 Project Description

The system takes a car dataset as input, processes features like year, fuel type, kilometers driven, and price, and predicts the selling price using a trained Machine Learning model (Linear Regression).

🚀 Features
Upload car dataset (CSV file)
Automatic data cleaning and preprocessing
Feature selection for prediction
Data scaling and encoding
Train/Test split (80% / 20%)
Prediction using Linear Regression
Actual vs Predicted price comparison
Graph visualization using Matplotlib

🧠 Tech Stack
Python
NumPy
Pandas
Streamlit
Matplotlib
Scikit-learn
